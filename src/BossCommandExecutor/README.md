# BossCommandExecutor 击杀boss自动执行命令插件

> 作者：星梦XM | 版本：2.0.0 

## 插件简介

**BossCommandExecutor** 当玩家召唤并击败指定的Boss后，插件会自动执行预设的命令序列，并提供伤害排行、浮动文本显示等增强功能。

## 支持的占位符

在命令、广播消息和浮动文本中可以使用以下占位符：

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `{boss}` | Boss名称 | "血肉墙" |
| `{boss.name}` | Boss名称（同{boss}） | "血肉墙" |
| `{boss.id}` | Boss ID | "113" |
| `{boss.type}` | Boss类型 | "113" |
| `{time}` | 当前时间 | "14:30:25" |
| `{date}` | 当前日期 | "2024-01-15" |
| `{count}` | 执行的命令数量 | "5" |

## 伤害排行功能

### 显示内容
伤害排行包含以下信息：

1. **基础信息**：
   - Boss名称
   - 参战玩家数量/在线玩家总数
   - 战斗用时（自动格式化：秒/分/时）
   - 仇恨目标（多人战斗时）

2. **Boss属性**：
   - 生命值上限
   - 攻击力
   - 防御力

3. **伤害统计**：
   - 总伤害
   - 玩家总伤害
   - 减益/陷阱伤害

4. **玩家排名**：
   - 排名序号
   - 玩家名（按队伍颜色显示）
   - 伤害百分比
   - 总伤害
   - 秒伤(DPS)
   - MVP标记（DPS最高者）
   - 最后一击标记

5. **未参战玩家**：显示在线但未参战的玩家列表

### 配置项说明

#### 全局配置
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| **插件开关** | bool | true | 控制插件是否启用 |
| **命令执行延迟(毫秒)** | int | 100 | 命令之间的执行间隔，防止刷屏 |
| **广播执行结果** | bool | true | 是否向全服广播命令执行结果 |
| **广播消息格式** | string | 详见配置 | 广播消息的格式，支持占位符 |
| **广播颜色** | object | R:50, G:205, B:50 | 广播消息的颜色（RGB格式） |
| **控制台提示** | bool | true | 是否在控制台显示命令执行成功提示 |
| **自动广播BOSS伤害排行** | bool | true | 是否自动广播Boss伤害统计信息 |

#### Boss命令配置
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| **Boss名称** | string | 无 | Boss的显示名称 |
| **Boss ID列表** | array | 无 | 该Boss对应的NPC ID数组 |
| **需要被召唤** | bool | true | 是否只对被召唤的Boss生效 |
| **记录执行次数** | bool | true | 是否记录该Boss的命令执行次数 |
| **执行次数** | int | 0 | 当前已执行的次数（自动记录） |
| **已首次击杀** | bool | false | 是否已完成首次击杀（自动记录） |
| **首次击杀命令** | array | 无 | 首次击杀时执行的命令列表（只执行一次） |
| **常规执行命令** | array | 无 | 每次击杀都会执行的命令列表 |
| **广播结果** | bool | true | 是否广播该Boss的命令执行结果 |

#### 浮动文本配置
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| **Boss名称** | string | 无 | 对应的Boss名称 |
| **Boss ID列表** | array | 无 | 对应的Boss ID数组 |
| **文本内容** | string | 无 | 浮动显示的文本内容，支持占位符 |
| **文本颜色** | object | R:255, G:255, B:255 | 浮动文本的颜色（RGB格式） |
| **显示时长(秒)** | int | 5 | 浮动文本显示的时间长度 |
| **是否启用** | bool | true | 是否启用该浮动文本 |
