# CustomDeathMessages - Custom Death Messages

- **Authors**: Eustia, 星梦

## Features
- Full death attribution: PVP, NPC/Boss, Projectile, Environment (fall, drowning, lava, poison, 21 types)

## Commands

| Syntax | Permission | Description |
|--------|:---------:|:-----------:|
| `/cdm reload` | `customdeathmessages.reload` | Reload config |
| `/cdm debug` | `customdeathmessages.reload` | Debug death attribution |
| `/cdm reset <player>` | `customdeathmessages.reload` | Reset player death count |
| `/cdm reset all` | `customdeathmessages.reload` | Reset all death counts |
