# PixelArtPlus - 像素画增强插件

基于 PixelArtPlugin 改进的高质量地图画生成插件。

## 核心改进

1. **综合调色板** - 从 `TerrariaTileMapColors.xlsx` 提取 27,965 种 (物块/墙体 × 颜料) 实际渲染颜色，匹配更精准
2. **Floyd-Steinberg 抖动** - 误差扩散算法使色彩过渡平滑，消除色阶断层
3. **颜料修正** - 直接匹配 (物块ID, 颜料ID) 组合，一步到位
4. **重力物块排除** - 自动排除沙、泥沙等受重力影响的物块
5. **亮度/饱和度调节** - 支持调色预处理
6. **智能混合模式** - 自动选择墙体或虚化物块获得最佳颜色匹配

## 命令

| 命令 | 说明 |
|------|------|
| `/pa list` (l) | 列出可用图片 |
| `/pa spawn <索引> [选项]` (s) | 物块模式 |
| `/pa wall <索引> [选项]` (w) | 墙体模式（推荐） |
| `/pa blend <索引> [选项]` (b) | 混合模式 |
| `/pa cancel` (c) | 取消当前操作 |
| `/parefresh` | 刷新图片列表 |

## 选项

| 选项 | 简写 | 说明 |
|------|------|------|
| `<宽> <高>` | - | 目标尺寸 |
| `scale:<N>` | - | 缩放（如 scale:0.5） |
| `bright:<N>` | - | 亮度 (默认1.0) |
| `sat:<N>` | - | 饱和度 (默认1.0) |
| `dither` | d | 启用 Floyd-Steinberg 抖动 |
| `paint` | p | 使用颜料修正颜色 |
| `nolight` | nl | 不放置照明墙 |
| `noclear` | nc | 不清除区域 |
| `ghost` | g | 虚化物块（可穿过） |
| `autofill` | af | 自动填充未成功放置的格子 |

## 使用示例

```
/pa wall 0 dither paint autofill
（墙体模式，第0张图，抖动+颜料+自动填充，推荐最清晰方案）

/pa spawn 0 scale:0.5 nolight ghost
（物块模式，缩放50%，无照明墙，虚化可穿过）

/pa blend 0 sat:1.2 bright:1.1
（混合模式，饱和度120%，亮度110%）
```

## 图片存放

将图片放入 `{TShock保存目录}/PixelArtImages/`

支持的格式: png, jpg, jpeg, bmp, gif

## 技术说明

- 颜色匹配算法：加权 RGB 距离 + 饱和度/亮度惩罚
- 颜料调色板：27,965 种 (类型ID, 颜料ID) → RGB 组合
- 抖动算法：Floyd-Steinberg 误差扩散 + 蛇形扫描
- 最大图片尺寸：1000×1000（超出自动居中裁剪）
- 排除的物块类型：重力物块（沙/泥沙）、液体、藤蔓、树干等
